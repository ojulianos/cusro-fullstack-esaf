Sobre os iframes, eles podem abrir inúmeras possibilidades.

O iframe tem como padrão o parametro src para informar qual página deve ser exibida assim que carregado.

Para enviar um conteúdo de uma tag a para uma tag iframe é necessário primeiro definir um nome para o frame usando o parametro name, então na tag a você deve informar com o parametro target o nome do frame onde será carregado o conteúdo.

Exemplo: 

                        
<a href="pagina.html" targe="mostrarPaginas">Abrir Página</a>
<iframe 
    src="home.html" 
    name="mostrarPaginas" 
    width="100%" 
    height="100%" 
    frameborder="0">
</iframe>

Ainda sobre os iframes...

Você pode colocar um frame dentro de outro frame, como nesse exemplo, onde essa página é um frame e o vídeo ao lado é outro.

Exemplo: 

                            
<iframe 
    width="500" 
    height="500" 
    src="https://www.youtube.com/embed/RV0LBPnfDLU?si=AlPvoV8lgk1eb0CN"
    title="YouTube video player"
    frameborder="0"
    allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share"
    allowfullscreen>
</iframe>
